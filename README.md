# User Registration Form Project

This is a professional user registration form built with **Pure HTML5, CSS3, and Vanilla JavaScript**.

## Features
- **Real-time Validation:** Checks inputs as you type.
- **Responsive Design:** Matches Figma specifications and works on all screen sizes.
- **Advanced Security Logic:** - Password must be at least 8 characters.
  - Includes numbers or symbols.
  - **Critical:** Password cannot contain the user's name or email address.
- **Smart Submit Button:** Remains disabled until all fields are perfectly validated.

## Technologies Used
- **HTML5:** Semantic markup.
- **CSS3:** Flexbox and modern layout techniques.
- **JavaScript:** ES6+ DOM manipulation and Regex validation.
